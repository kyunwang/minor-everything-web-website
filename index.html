    As much native HTML, CSS & JavaScript as possible. We will be making a web app from scratch without frameworks or unnecessary libraries. The deliverable is a modular made prototype for an single page web app. Your will fetch data from an external API, which when needed will be manipulated and showed in the app. You will be learning how to structure your code and how to make an Web App from Scratch.     • Progressive Enhancement
    • Browser Compatability

    One of the greatest principles of the web is that it is for everyone, really. Anyone with an computer and browser should be able to make use of the web. Our web is not a controlled (development) environment. You can be sure that nobody will see the same as what you will be seeing in your browser. Of course there are technical constrains too. Take the measurements of the browser or device for instance, the way of interaction, quality of the hardware and network. And there are people of all kinds.

    During this course you will be learning how to build great, robust and accessible websites. You will learn about Progressive Enhancement, Feature Detection and Fallbacks. The web is for everybody and we will teach you how you can make that possible.
